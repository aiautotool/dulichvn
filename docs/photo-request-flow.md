# Photo Request Flow

Target flow:

1. Traveler taps **Request Current Photo — $0.30**.
2. Payment is held.
3. Nearby helpers receive the job.
4. One helper accepts transactionally.
5. Helper captures required shots using device camera.
6. App stores timestamp and GPS metadata.
7. Requester reviews submitted photos.
8. Job completes and helper payout is released.

Verified live photos must not come from the normal gallery in production.
